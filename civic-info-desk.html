About this prototype

The Civic Information Desk is a conversational AI chatbot for public service information access, built for TCS AI Friday Season 2. It routes citizen questions to a specialized assistant per department, with voice conversation ("Talk Mode"), a live nearby-office map, and a simulated appointment booking flow with reminders.

Important: what's real vs. simulated

The AI answers are real (generated live by the connected model). The nearby-office map uses your real location against live Google Maps data. Appointment booking is a simulation — no booking is actually created with any real government office. Reference numbers are generated locally and stored only in this browser. Reminders only appear when this page is open on the appointment day; there is no server-side notification system.

How it works

Each question is sent to a Flask backend, which calls a large language model with a department-specific persona and the selected language. The frontend never talks to the model directly — the API key stays server-side.
