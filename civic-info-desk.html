About this prototype

The Civic Information Desk is a conversational AI chatbot for public service information access, built for TCS AI Friday Season 2. It routes citizen questions about permits, tax payments, and local regulations to a specialized assistant per department, with an animated routing sequence, a scenario-check mode for planning ahead of filing, voice input, document context, multilingual replies, and a persistent record of past inquiries.

How it works

Each question is sent to a Flask backend, which calls a large language model with a department-specific persona, the selected language, and any attached document context. The frontend never talks to the model directly — the API key stays server-side.

Scope

This is a working prototype, not a production civic service. Fees, dates, and procedures are described in general terms and should be confirmed with the relevant department before acting on them.
